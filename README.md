# Nexus
Nexus est la première plateforme d'automatisation desktop basée sur l'IA qui transforme vos instructions en actions concrètes. Au lieu de configurer des workflows complexes ou d'écrire du code, vous décrivez simplement votre besoin en langage naturel, et notre AI comprend le contexte visuel de votre écran pour exécuter les tâches automatiquement.
